## ⚙️ Technical Details

### Sensor Calibration
| 항목 | 값 / 방법 |
|------|----------|
| 센서값 범위 | 디지털 입력: HIGH(1) = 흰색/라인 없음, LOW(0) = 검정색/라인 감지 |
| 임계값 설정 | 아날로그 센서 사용 시 `analogRead()` 평균값 기준 400~600 사이를 임계값으로 설정 가능. 

현재는 디지털 입력이므로 LOW/HIGH 단순 판별 |
| 노이즈 처리 | - 센서 값 읽기 후 `delay(2~5ms)`로 안정화<br>- 여러 번 샘플링 후 평균값 사용<br>- 필요 시 소프트웨어 디바운싱 적용 |

---

### Speed Optimization
| 항목 | 내용 |
|------|------|
| 기본 속도(직진) | 좌: 173 / 우: 153 (안정적 주행 기준) |
| 코너링 전략 | - 좌회전: 좌측 모터 속도 +34 보정<br>- 우회전: 우측 모터 속도 +36 보정<br>- 급커브 시 속도 차이를 크게 두어 선회 반경 축소 |
| 감속 조건 | - 센서가 라인을 놓쳤을 때(모두 HIGH) → 속도 감속 후 탐색<br>- 부스터 종료 후(1.5초 이후) → 기본 속도로 복귀<br>- 코너 진입 시 속도 차등 적용으로 자연스러운 감속 효과 |
